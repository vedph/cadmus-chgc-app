## Docker

🐋 Quick Docker image build:

1. update version in `env.js` and `ng build`.
2. `docker build . -t vedph2020/cadmus-chgc-app:0.0.1 -t vedph2020/cadmus-chgc-app:latest` (replace with the current version).
